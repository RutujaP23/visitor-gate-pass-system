## üîÑ System Workflow

1. Visitor/Employee submits a gate pass request via the mobile app
2. Admin reviews and approves or rejects the request via the web dashboard
3. Approved visitors receive a unique QR code via email
4. Security scans the QR code at entry and exit points
5. Entry and exit times are logged automatically in the database
